A — 列表 + 右侧抽屉详情

列表页展示所有需求，点击行或"详情"按钮从右侧滑出抽屉，在抽屉内完成上传、触发拆解/生成、下载等所有操作。无需跳转页面。

优点

  • 操作流畅，无页面跳转
  • 列表始终可见

缺点

  • 抽屉空间有限，内容多时拥挤

B — 列表 + 下方展开详情（推荐）

C — 列表 + 独立详情页（跳转）

列表页点击进入独立的详情页，详情页有完整的步骤进度条（上传→拆解→生成→下载），内容区域最大化展示测试用例。

优点

  • 详情页空间最大
  • 步骤流程最清晰

缺点

  • 需要页面跳转
  • 需要额外路由配置
